# Trek

(*Alpha version of a CLI/ncurses explorer for Nomad clusters.*)

Trek is an interface to explore Nomad clusters.

## SETUP

### Binary Release

Get to revisions, and download a binary.

### From Source

    git clone https://github.com/franckverrot/trek.git
    cd trek
    make trek


## USAGE

### CLI

    ./trek <task name>

### ncurses UI

    ./trek
